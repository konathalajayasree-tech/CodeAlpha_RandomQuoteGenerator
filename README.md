# Random Quote Generator

## Description

Random Quote Generator is a simple Python application built using Tkinter. The application displays a random motivational quote along with the author's name whenever the user opens the app or clicks the "New Quote" button.

## Features

* Displays a random quote on startup
* Shows the author's name clearly
* Generates a new random quote on button click
* Clean and user-friendly interface
* Built using Python and Tkinter

## Technologies Used

* Python
* Tkinter
